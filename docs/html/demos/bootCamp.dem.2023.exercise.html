These exercises are designed to help you get creative with your coding design and project choice. They should help you see the possibilities for exciting and original sketches.

1. Choose any of the sketches you have worked on this week and save a copy in a folder 'projectIdeas'.

Try modifying with the sketch so that it does something slightly different. This might involve changing the colours or shapes shown, or the number of features in the sketch, or how it responds to mouse movement or key presses. How you modify the sketch is up to you. Try to get creative, and don't forget to have fun!

Show your modified sketch to someone else in your lab class and ask them to make suggestions about how you might modify it further. Can you make the changes they suggest? Does it give you ideas about how you might change the sketch in a new direction?


2. Browse Processing's Examples pages. Copy one of the examples from those pages and try modifying it, just as you did for the previous task. Does this suggest new possibilities for your own project?


3. Are there any libraries there that might help you use Processing to create an interesting project? By following the documentation / tutorials for a library of your choice, develop a simple sketch that makes use of one of these libraries.




4. In your own time, watch the Hello World! Processing documentary.



Does this give you any ideas for your own projects?
Remember that successful coding is about creativity as well as logical and structured thought.
