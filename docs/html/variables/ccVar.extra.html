Digital Timer

Code a sketch that presents a digital timer, showing the number of hours, minutes and seconds that have elapsed since the sketch started running. Some useful things to know:

  • Processing has a built-in integer variable frameCount which keeps track of how many times draw() has been executed since the sketch started running. And by default, draw() is executed 60 times per second.
  • For integers a and b, a / b evaluates to the integer division of a by b.
    For example, 4 / 5 = 0, 7 / 5 = 1, 16 / 5 = 3.
  • For integers a and b, a % b evaluates to the remainder of the integer division a / b.
    For example,
    10 % 5 = 0, as 5 is a factor of 10.
    7 % 5 = 2, as 2 remains when 7 is divided by 5.
    16 % 5 = 1, as 1 remains when 16 is divided by 5.
    16 % 4 = 0, as 4 is a factor of 16.
    4 % 5 = 4.
  • textWidth(s) is the display width in pixels of the String s.
  • nf() can be used to format a number as a String with a fixed number of digits.
  • For Strings a and b, a + b is the string obtained by joining a and b end to end (string concatenation).
    And if a is a String but b is not a String, then b will first be converted to a String, before concatenating with a.
