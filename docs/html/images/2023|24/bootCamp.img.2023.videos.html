Some short videos that introduce you to the materials that we have developed for this session on Images, and some of the capabilities of the PImage in Processing.

Click the images to the right to watch them through and try out the ideas.

The imageGrid application is available in the Images lecture. It is intended to help you explore the overall concept of the 'image' as a structured set of numbers representing colour defined according to their red, green and blue content.
Be sure to spend time experimenting with it.

You will notice that the API used to code in Processing is a little old, and both Moodle and the Processing documentation have had something of a re-vamp since the videos were recorded. Be reassured that the concepts and the code are exactly the same, so don't let this put you off. Everything changes and nothing changes!

Images Introduction

A Quick Introduction to the Images Resources - 18:45

Here Jason introduces the resources that you should use in this session.

He produces a quick example sketch that reveals what he was up to at the weekend and shows how the key concepts might be used together:

  • PImage - a type of variable that you can use to store, manipulate and display images in your sketches;
  • loadImage() - to load an image from file into a PImage object that you can then use in your Processing sketch;
  • save() - a Processing method that will create an image from whatever is displayed on the canvas;
