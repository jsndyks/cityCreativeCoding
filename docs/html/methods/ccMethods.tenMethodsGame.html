1. method 01 : Can you write a method with no content that does nothing?

Use this code as the basis of your sketch, and fill in the eight arguments to quad() that will complete the task.

      void setup()
      {
        size(600, 600);
      }
      
      void draw()
      {
        background(255);
        stroke(32);
        noFill();
      
        float xPos   = 100;
        float yPos   = 100;
        float sSize  = 100;
        float sHalf  = sSize/2;
      
        quad(
          // -- add the eight arguments here ...
        );
      }
Solution - Catch up Here! 
            quad(
              // -- add the eight arguments here ...
              xPos - sHalf, yPos - sHalf,
              xPos + sHalf, yPos - sHalf,
              xPos + sHalf, yPos + sHalf,
              xPos - sHalf, yPos + sHalf
            );
