Exercises & Extras

These exercises give you practice using loops by setting some additional extra challenges for those who found the core material straightforward.
They should help you develop your understanding of how these are coded in Processing and help you practice and use your skills in making repeated tasks more efficient by placing them inside these structures.

Note that these are optional exercises - be sure that you have completed and understood and discussed the core exercises first. Only attempt these exercises if you have completed everything in Loops : Core and want more!

Interactive Trees

Adapt your three rows of trees sketch once again to make it interactive.

Build on the underground sign sketch to check whether the mouse is within the trunk or leaves of the tree and change the colour of any tree you are 'touching'.

TIP Calculating whether a point is inside or outside a non-circular ellipse is a little more complicated than for a circle.
But we can use add a method do the calculation - try this one: 

// -- Method will return true if position (x,y) is within an ellipse(cx,cy,w,h), otherwise false.

boolean isInEllipse(float x, float y, float cx, float cy, float w, float h) 
{
  return (x-cx)*(x-cx)/(w*w/4) + (y-cy)*(y-cy)/(h*h/4) < 1;
}
