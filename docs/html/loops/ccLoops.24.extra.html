TIP Calculating whether a point is inside or outside a non-circular ellipse is a little more complicated than for a circle. But we can use our own method do the calculation: 

// Will return true if position (x,y) is within an ellipse(cx,cy,w,h), otherwise false.
boolean isInEllipse(float x, float y, float cx, float cy, float w, float h) 
{
  return (x-cx)*(x-cx)/(w*w/4) + (y-cy)*(y-cy)/(h*h/4) < 1;
}
