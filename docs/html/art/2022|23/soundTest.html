To play sounds with Processing, you must have downloaded the Processing Sound Library. If you haven't already, you can do so by opening Processing's Sketch -> Import Library -> Manage Libraries menu.

In the new window, choose the 'Libraries' tab and select Sound by the Processing Foundation (you may find it easier to find by entering 'sound' in the search box in this window).

Then click the Install button once selected.

You only need to do this once, after which you should be able to use sound in any sketch you write.

Testing Sound

The following sketch should allow you play a sound when the space bar is pressed: 

    import processing.sound.*;   // Import the library that does the sound handling.

    SoundFile mySample;

    void setup()
    {
      size(300, 200);
      mySample  = new SoundFile(this,"XXXX.wav");
    }

    void draw()
    {
      background(255);
      fill(0);
      textAlign(CENTER,CENTER);
      textSize(18);
      text("Press space to test", width/2, height/2);
    }

    void keyPressed()
    {
      if (key == ' ')
      {
        mySample.play();
      }
    }
Before this can work, you will need to download a sample file containing the sound to play and drag it into your sketch. You should then change the XXXX above to match the name of a sample you have stored on your computer.
