1. Processing Setup

Download and install the latest version of Processing from The Processing Foundation :
  • https://processing.org/download/

2. Hello Processing

Spend an hour or so working your way through Dan Shiffman's impressively enthusiastic, and very useful, introduction to Processing :
  • https://hello.processing.org/

Dear Data

Spend an hour or so looking at the Dear Data materials that we provide and recommend.

Of course you could spend much more time on this, if it piques your interest than keep reading.

Data Diary

Begin thinking about your Data Diary and perhaps start collecting some data.

There is some guidance here to help.
