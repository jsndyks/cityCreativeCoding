Moving Sign

Modify your underground sketch so that it moves or changes size in response to the mouse.

Hint: As with the core tasks you will need to use mouseX and mouseY but this time you may need to add or subtract numbers from these values to draw some of the shapes in the right place.

Make your sketch original - try changing something to give it some character that others may not have thought of.

Use a creativity framework, such as the SCAMPER technique to help you do so.
