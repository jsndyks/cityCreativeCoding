Create a new sketch with just a setup() method. In setup() declare and initialise some variables to represent the following: 

number of apples: 12
number of bananas: 7
number of oranges: 6
Use a simple arithmetic expression on the variables to find the total number of fruit and display the result using println().


Travelling circle.
  1. Write a sketch that draws a circle in the centre of a square window. Use two variables x and y to specify the position of the circle:
    ellipse(x, y, 60, 60);
    Remember you have to declare those variables (both with type float) at the top of your sketch and you should initialise them with appropriate values inside setup() so that your circle is positioned correctly when your sketch starts.
    Run your sketch to check it behaves as expected.
  2. Now add assignment statements to the end of your draw() method so that the values of x and y both increase by 1 every time draw() is executed (remember, draw() will be executed automatically, 60 times every second while your sketch is running).
    What effect do you think this will have? Run your sketch to see if you were right.
  3. Modify your sketch so that the circle travels diagonally upwards towards the top-left corner of the window.


Travelling circles. Make a copy of your Travelling circle sketch. Modify this new sketch so that it draws multiple circles instead of just one. Make all your circles start in the centre of the sketch window but move them in different directions (e.g. four circles could move each to one of the four corners).

Create some colour variables and use them to fill each circle with its own colour.


How many variables did you use in the travelling circles task? Because of the symmetry of moving four circles each to their own corner, it can be achieved with only a single variable (plus a bit of arithmetic).

Code a digital timer, showing the number of hours, minutes and seconds which have elapsed since the sketch started running. Some useful things to know:
  • Processing has a built-in integer variable frameCount which keeps track of how many times draw() has been executed since the sketch started running. And by default, draw() is executed 60 times per second.
  • For integers a and b, a / b evaluates to the integer division of a by b. For example, 4 / 5 = 0, 7 / 5 = 1, 16 / 5 = 3.
  • For integers a and b, a % b evaluates to the remainder of the integer division a / b. For example, 4 % 5 = 4, 7 % 5 = 2, 16 % 5 = 1.
  • textWidth(s) is the display width in pixels of the String s.
  • nf() can be used to format a number as a String with a fixed number of digits.
  • For Strings a and b, a + b is the string obtained by joining a and b end to end (string concatenation). And if a is a String but b is not a String, then b will first be converted to a String, before concatenating with a.
