Dear Data Diary

We want you to collect data in similar fashion to the Dear Data Project over the term.

We will show you how to use Processing to create a Dear Data Diary that explores some aspect of the World that is interesting to you, filling your code with data and using code to express your data in ways that are meaningful to you.

With luck, you will learn about yourself and some aspects of The World as you embark upon some discovery by depicting your data in creative ways.

The Topic and Data that you collect must be:

  • ORIGINAL - the topic must be original and unique - use the creativity techniques that we have touched upon, like SCAMPER;
  • MEANINGFUL - the topic must be something that interests and is perhaps important to you;
  • ANONYMOUS - the data must not reveal identifying personal information;
