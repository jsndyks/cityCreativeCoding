If you read through these notes, attempt the tasks and exercises and use the resources that we have made available, then you are expected to be able to:

  • Drag and drop data files into your Processing code window to make them accessible to your sketches;
  • Create a sketch to read a text file using loadStrings();
  • Split a line of text into tokens representing individual words or numbers with split();
  • Rescale a number from one range to another using the map() command;
  • Use the class command to define an object that captures data relating to your observations;
  • Depict data that is stored in a digital data file on the canvas;

We're going to look at ways of getting data into your sketches here, and will build upon some of the ways of collecting data that you may have embarked upon. A few are described on the Data Collection page, and we will use some of these to demonstrate.

1. Dear Data Sketch

We have asked to to produce a Dear Data Sketch.

This should be ...

A Processing sketch
that depicts a data set that you have collected over time
that explores some aspect of the World
that is interesting to you

We hope that you will do this ...

in creative ways

And that the process ...

helps you learn about yourself, data & The World around you.

Maybe just a little!

Those are the key requirements, but additionally ...
We would like the sketch to be ...

inspired by the Dear Data Project.

We would like to see you using the coding constructs and capabilities that we have introduced, including ...

Methods, Classes (and Objects), Interaction, and perhaps Sound and / or Video

We also encourage you to ...

be expressive;
find your voice;
experiment;
celebrate your mistakes;
don't worry about anything;
take this wherever you want it to go;
have fun;
not go crazy
(by which we mean - put a reasonable amount of work into this, and make sure that other commitments, sleep, nutrition, relationships, exercise, keeping up wht The Great British Bake-Off as it reaches its unimaginably tense and exciting denouement, are not compromised);

That's it - see what you can do.
