1. Modify the name list sketch given in the lectures (reproduced below) so that the remaining names are removed when you press a key corresponding to the first letter of each name. Modify if further so that pressing shift and the relevant key, adds names back to the list (e.g. pressing shift-S should add all the names beginning with 'S'). 

ArrayList<String> names = new ArrayList();

void setup()
{
  size(300,350);
  fill(0);
  textSize(22);

  names.add("Jon");
  names.add("Jason");
  names.add("Seb");
  names.add("Radu");
  names.add("Eva");
  names.add("Andy");
  names.add("Dympna");
  names.add("Ross");
  names.add("Simone");
  names.add("Charlie");
  names.add("Jo");
}

void draw()
{
  background(255);
  float yPos = 30;

  for (String name : names)
  {
    text(name,50,yPos);
    yPos = yPos+30;
  }
}

void keyPressed()
{
  if (key == 'j')
  {
    names.remove("Jon");
	names.remove("Jason");
    names.remove("Jo");
  }
  else if (key == 's')
  {
    names.remove("Seb");
    names.remove("Simone");
  }
}
Hint: You can distinguish between a key press and a shift-key press by simply testing for the lower-case and upper-case versions of the key, e.g. 

if (key == 'a')
{
  // Do something in response to the lower-case key press.
}
else if (key == 'A')
{
  // Do something in response to the upper-case key press.
}



2. Modify the circle grid sketch from the lecture notes (reproduced below) so that the fourth row of circles is displayed in a different colour. 

void setup()
{
  size(400,400);
  noStroke();
  fill(180,50, 50, 100);
}

void draw()
{
  background(255);

  for (int row=0; row<10; row = row+1)
  {
    for (int col=0; col<10; col = col+1)
    {
      circle(20 + col*40, 20 + row*40,30);
    }
  }
}

Modify the sketch further so that it instead highlights the fourth column.

4. Create a sketch containing the setup, draw, mousePressed, and mouseReleased methods. Declare two global float arrays called xps, yps and initialize them to have the same size. In the setup method fill the two arrays with random values such that pairs of xps and yps values at the same index represent the coordinates of points that lie within the sketch bounds. In the draw method, draw circles of size 30 at those positions.

Add code to allow the user to pick a specific circle and drag and drop it anywhere within the sketch. To do so you may want to add a global variable of type int called pickedCircle which should be set to a negative value if no circle is picked and a value corresponding to the index of a picked circle, in case a circle was indeed picked.

Add two more global arrays, one called diameters (of type float) and one called colours (of type color) and initialize them to have the same size as the xps and yps arrays. Fill the diameters one with random values and the colors one with one of two colours, red and blue, chosen randomly. Given the drag-and-drop functionality the user should be able to separate the randomly positioned circles into separate red and blue groups (ex: top half of the screen vs. bottom half of the screen). As a challenge you may wish to consider writing a method to test automatically if the user accomplished this goal.

Change the arrays to ArrayLists and change the implementation of the mousePressed method so as to allow the user to remove all circles of a certain colour.
